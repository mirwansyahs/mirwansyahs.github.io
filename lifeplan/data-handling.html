Retensi Data

Data Anda disimpan selama Anda memiliki akun aktif. Jika Anda menghapus akun, semua data akan dihapus dalam waktu 30 hari. Backup terenkripsi mungkin tersimpan hingga 90 hari untuk keperluan pemulihan darurat, setelah itu akan dihapus permanen.

Keamanan Tambahan

Untuk melindungi data Anda, kami merekomendasikan:

  • Gunakan password yang kuat dan unik
  • Aktifkan autentikasi dua faktor jika tersedia
  • Jangan bagikan kredensial akun Anda
  • Logout dari perangkat yang tidak digunakan
  • Perbarui aplikasi ke versi terbaru
